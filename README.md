# MIA-mini-project-Ed22B009-14
Brain tumor segmentation using a 3D U-Net model trained on the BraTS 2021 dataset. Includes multimodal MRI preprocessing, tumor-aware patch extraction, mixed-precision + multi-GPU training, Dice and Hausdorff evaluation, and interactive 3D visualizations. Achieved a validation Dice score of 0.7792 in 5 epochs using dual Tesla T4 GPUs.
